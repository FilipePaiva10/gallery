- ✔️ [TypeScript](https://www.typescriptlang.org/)

- ✔️ [uuid](https://www.npmjs.com/package/uuid)

## Clonar o repositório
```bash
$ git clone https://github.com/FilipePaiva10/gallery.git
```

## Acessar Pasta
```bash
cd gallery
```

## Instalar dependências
```bash
npm install
```

## Rodar Projeto
```bash
npm start
```
